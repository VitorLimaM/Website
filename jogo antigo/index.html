Pong Game

Clássico jogo de ping pong para 2 jogadores!

Jogador 1: 0 | Jogador 2: 0

Controles

Jogador 1 (Esquerda)

↑ W ou ↑ (Seta para cima) - Mover para cima

↓ S ou ↓ (Seta para baixo) - Mover para baixo

No mobile: Use os botões na tela

Jogador 2 (Direita)

↑ ↑ (Seta para cima) - Mover para cima

↓ ↓ (Seta para baixo) - Mover para baixo

No mobile: Use os botões na tela

Primeiro a marcar 5 pontos vence! Ajuste a velocidade pressionando as teclas.
